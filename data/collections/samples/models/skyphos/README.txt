3D model of an Etruscan skyphos from the the Archeological Museum of Fara in Sabina (Italy). This  model has been made by CNR-ISPC within  "Innova Patrimonio", a research project funded by Regione Lazio. 
The skyphos has been digitised using photogrammetric techniques with the software 3DFZephir. Author: Daniele Ferdani (CNR ISPC)

This dataset is intended to provide an example for educational purposes. Use or reuse of the assets for other projects needs a written permission by the mentioned authors.
